Success or fail?

Define a function check_success(url) that, given a URL, sends out a GET request and returns "Success" if it succeeds and "Fail" otherwise.

Remember that the status codes between 200 and 400 indicate that the request succeeded, while all the codes starting with a 4 signalize an error.



Memory limit: 256 MB
Time limit: 30 seconds

Caution

You may see errors in your code or in execution results because of missing context. Don't mind those; simply write your solution and click Check.
