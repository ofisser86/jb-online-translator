Let's say there's a script process_four_numbers.py. It takes exactly four arguments, all of which are numerical.

Inside this script, the first thing to do it is to get the arguments from the command line. Below, write a program that does so and prints them as a list.

Note that interaction with the command line takes place in the computer itself so we can't check this task directly. So let's make it this way: instead of sys.argv, use the predefined list args. It contains what you would otherwise get from sys.argv, and you should just write what you would do next.

For example, if args = ["process_four_numbers.py", "1", "2", "3", "4"], then your output should be [1, 2, 3, 4] with the list elements as numbers.
